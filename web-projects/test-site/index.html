ゲームの準備

  1. 鉱物を、右表の個数分、袋の中に入れて良く混ぜます。
  2. 各プレイヤーはそれぞれベーシック魔法陣を2枚ずつ、絵柄が円になるように向きを合わせて手前に並べます。
  3. 最近闇市に行ったプレイヤーがスタートプレイヤーです。
    スタートプレイヤーから時計回りに、1周するまで順番に、袋から3個ずつランダムに鉱物を取り出し、それぞれの手元に置きます。
  4. アドバンスド魔法陣6枚をすべて裏向きにして良く混ぜ、場の近くに置きます。これを山札と呼びます。
